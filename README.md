Posgrado - Demo frontend

Instrucciones rápidas:

1. Ir a la carpeta `frontend`.
2. Ejecutar `npm install`.
3. Ejecutar `npm run dev`.
4. Abrir `http://localhost:5173`.

Usuarios demo:
- admin / admin  (conducción)
- docente / docente (docente)
- est1 / est1 (estudiante)

La demo usa localStorage para persistir datos. Si querés reiniciar los datos, borrar la clave `posgrado_demo_data` del localStorage.
